# Changelog

All notable changes to VoiceFlow will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-06-15

### Added
- Initial release of VoiceFlow
- Real-time speech recognition using Web Speech API
- Light and dark theme support with persistent preference storage
- Responsive user interface for desktop and mobile devices
- Text export and clipboard copying functionality
- Error handling and user-friendly error messages
- Visual recording indicators and status updates
- Accessibility features for screen reader compatibility
- Vercel Analytics integration
- Comprehensive documentation and README
- Contributing guidelines for developers
- MIT License for open-source distribution

### Features
- ⚡ Real-time speech-to-text conversion
- 🎨 Modern dark/light theme toggle
- 📱 Fully responsive mobile-first design
- 🎯 Clean, intuitive user interface
- 🌍 Multi-language support
- 📋 Easy copy and export options
- ♿ Accessible UI components
- 🚀 Deployed on Vercel CDN

### Technology Stack
- Next.js 16.2.6
- React 19
- TypeScript 5.7
- Tailwind CSS 4.2
- shadcn/ui components
- Lucide React icons

- **MAJOR (X.0.0)**: Incompatible API changes

- **PATCH (0.0.X)**: Bug fixes, backwards compatible
